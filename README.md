# Shodan Geo Search

Aplicacao web com backend em Node.js/Express e frontend em HTML, CSS e JavaScript vanilla para buscar dispositivos por geolocalizacao usando a API do Shodan.

## Como usar

1. Instale as dependencias com `npm install`
2. Copie `.env.example` para `.env`
3. Defina `SHODAN_API_KEY`
4. Rode com `npm start`
5. Abra `http://localhost:3000`
